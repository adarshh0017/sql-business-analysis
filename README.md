# 📊 Business Data Analysis using SQL

## 🔍 Overview

This project focuses on analyzing e-commerce business data using SQL. The goal is to extract meaningful insights such as sales trends, product performance, and customer behavior.

## 🛠 Tools Used

* MySQL
* SQL

## 📁 Dataset

* Order Details.csv
* List of Orders.csv

## ⚙️ Process

* Imported CSV data into MySQL database
* Created structured tables for sales and orders
* Performed data cleaning by fixing schema mismatches
* Wrote SQL queries to analyze business performance

## 📈 Key Analysis

* Total sales calculation
* Category-wise sales performance
* Top-selling products
* Profit analysis
* Loss-making products identification
* State-wise sales using JOIN
* Top customers based on spending

## 🧠 Key Insights

* Certain categories generate higher revenue
* Some products result in losses despite high sales
* Sales distribution varies across states
* Top customers contribute significantly to revenue

## 🚀 How to Use

1. Import datasets into MySQL
2. Run queries from queries.sql
3. Analyze results for insights
